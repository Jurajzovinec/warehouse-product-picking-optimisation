# Warehouse product picking optimization

### Prerequisites

- NodeJs v16+

### Initial setup

1. Install app using:

- `npm ci`

### Usage

1. Create `./env` in the root either manually or by running command below. Fill env variables:

- `cp .sample.env .env`

2. Start app using:

- `npm run start`

4. Test exposed endpoints
